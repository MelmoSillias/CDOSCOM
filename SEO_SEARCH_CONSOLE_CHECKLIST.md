# Checklist Search Console - CDOSCOM

## 1. Verification de propriete
1. Ouvrir Google Search Console et ajouter la propriete de domaine: https://cabinetdentaireousmanesow.cloud
2. Verifier la propriete via DNS (recommande) ou balise HTML.
3. Confirmer que la version https est bien la version principale.

## 2. Soumission du sitemap
1. Ouvrir la section Sitemaps.
2. Soumettre l'URL: https://cabinetdentaireousmanesow.cloud/sitemap.xml
3. Verifier le statut: "Recupere avec succes".

## 3. Controle robots et indexation
1. Verifier que https://cabinetdentaireousmanesow.cloud/robots.txt est accessible.
2. Verifier que robots.txt contient bien le lien du sitemap.
3. Ouvrir l'outil d'inspection d'URL pour 5 pages cles:
   - /
   - /services
   - /actualites
   - /actualites/{slug}
   - /contact
4. Verifier que l'etat est "URL sur Google" ou demander l'indexation.

## 4. Couverture et qualite d'index
1. Ouvrir Indexation > Pages.
2. Verifier l'absence d'erreurs severes sur les pages publiques.
3. Examiner les exclusions (duplication, noindex, redirections inattendues).

## 5. Ameliorations SEO
1. Ouvrir "Ameliorations" et verifier la detection des donnees structurees.
2. Corriger les erreurs JSON-LD signalees (si presentes).
3. Suivre Core Web Vitals mobile et desktop.

## 6. Suivi des performances (2 a 4 semaines)
1. Activer le rapport Performance (requete, pages, pays, appareil).
2. Noter une baseline jour 0:
   - Impressions
   - Clics
   - CTR moyen
   - Position moyenne
3. Comparer chaque semaine les pages cles: /services, /actualites, /contact.

## 7. Alertes et routine
1. Activer les notifications Search Console par email.
2. Planifier un controle hebdomadaire des erreurs d'indexation.
3. Re-soumettre le sitemap apres chaque lot de nouvelles actualites.
